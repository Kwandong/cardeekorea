# CardeeXeno™ Landing Page v3

애플 액세서리 브랜드 느낌을 목표로 재설계한 v3입니다.

## 실행
압축을 풀고 `index.html`을 브라우저에서 열면 됩니다.

## 스마트스토어 링크 교체
`index.html`에서 `https://smartstore.naver.com/`를 실제 링크로 바꾸세요.

## 영상 위치 조정
`css/style.css`의 아래 값을 조정하세요.

```css
.video-frame video { object-position: center 63%; }
```

제품을 더 아래로 보이게: 68%  
제품을 더 위로 보이게: 58%
